# Miasma

Miasma color scheme for VS Code, generated with [themer](https://github.com/mjswensen/themer) and tuned to match the Omarchy Miasma theme.

## Install

- Copy this folder into your VS Code extensions directory.
- Restart VS Code.

## Use

- Open the Command Palette.
- Run `Preferences: Color Theme` and select `Miasma`.
